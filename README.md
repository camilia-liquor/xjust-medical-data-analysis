# XJUST Medical Data Analysis Project

## 📋 Project Overview
This project aims to analyze medical datasets for disease prediction and data quality assessment.

## 🛠️ Tech Stack
- Python 3.9+
- DeepChecks (Data Quality Validation)
- Pandas & NumPy

## 📁 Structure
- `src/`: Source code for data processing
- `notebooks/`: Jupyter notebooks for exploration
- `datasets/`: Raw and processed data
- `reports/`: Generated analysis reports
